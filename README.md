# FUZZY
Fuzzy.py is a fuzzbunch wrapper but its made to look like the original fb.py
<h1>INSTALL:</h1><pre>git clone https://github.com/Eitenne/FUZZY.git && cd FUZZY && sudo python setup.py</pre>
<h1>RUN:</h1><pre>fuzzy</pre>
